# Social Network

### Getting started
- Create an account
- Log in with Facebook
- Seed database with db:seed

### Posts
- Create posts
- Create comments
- Like posts
- Like comments

### Friendships
- Request friendships
- Accept friendships
- Delete requests / friendships


### Site nav
- Feed is collection of friends' posts (and current user)
- Explore is collection of all users' posts
